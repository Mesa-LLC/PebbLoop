# PebbLoop
Ever wished being able to play Zuma or Puzz Loop on your color-screened Pebble? Now you can! Survive as long as you can in this addictive and fast-paced ball-shooting game! Shoot balls of the same color to eliminate them and collect precious points to beat your previous high score. But remember to aim carefully; every missed target will push you closer to defeat!

How to play? It’s so simple, fun and intuitive:

- UP: rotate the shooter counterclockwise 

- DOWN: rotate the shooter clockwise

- CENTER: shoot current ball

- BACK ONCE: pause

- BACK TWICE: exit

- WRIST FLICK: swap current and next ball in the shooter

Touchscreen, Speaker and Round Screen support planned for a later date. Claude was used, but the code was reviewed and debugged manually.
